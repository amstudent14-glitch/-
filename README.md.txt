✈️ 極光之旅打包清單 (Progressive Web App)這個專案是一個為極地旅行，特別是前往黃刀鎮 (Yellowknife) 觀賞極光而設計的打包清單工具。🎯 功能特色極地專屬清單: 包含為極端低溫（-30°C）準備的專業保暖裝備。自訂主題色: 可選擇您喜歡的主題顏色。進度追蹤: 實時顯示打包進度條。PWA 應用: 可直接將網站「安裝」到手機或電腦主畫面，像一個原生 App 使用（支援離線儲存進度）。💡 如何安裝為 App開啟您的手機瀏覽器 (Chrome, Safari, 或 Firefox)。輸入此清單的網址 (即 GitHub Pages 提供的網址)。在 Android (Chrome): 點擊右上角選單，選擇「安裝應用程式」。在 iOS (Safari): 點擊底部的「分享」按鈕，選擇「加入主畫面」。一旦安裝完成，它就會在您的主畫面上顯示「極光清單」的圖示，點擊即可開啟，並像 App 一樣全螢幕運行。📝 檔案結構index.html: 單一檔案應用程式主體（包含所有 HTML, CSS, JavaScript）。manifest.json: 應用程式安裝清單，定義 App 名稱和圖示。
